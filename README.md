# Prueba Técnica ALCA

Repositorio que contiene la resolución de la prueba técnica para el proceso de selección.

## Tecnologías utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Contenido
- Ejercicio 1: Limpieza y transformación de datos.
- Ejercicio 2: Análisis exploratorio y métricas.
- Ejercicio 3: Visualización y análisis de resultados.
